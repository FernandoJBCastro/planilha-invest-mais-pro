# planilha-invest-mais-pro
A Planilha Invest Mais PRO - Simulador Financeiro é uma ferramenta desenvolvida em Microsoft Excel com foco em educação financeira e simulação de investimentos de longo prazo. A planilha permite ao usuário projetar a evolução do seu patrimônio com base em renda, gastos, perfil de investidor, taxa de rendimento e tempo de investimento.
